# Magnify — downloads

Public download channel for **Magnify**, a calm, local-first desktop app that shows you what your
computer is actually doing — live CPU, memory, processes and network, with privacy and system
intelligence that runs entirely on your own machine.

Get the installer from the **[Releases](../../releases)** page.

- Latest: `magnify-v0.8.1`
- Installer: `Magnify_*_x64-setup.exe` — Windows x64, WebView2-based, ~7 MB.
